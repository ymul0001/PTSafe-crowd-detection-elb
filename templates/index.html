People Counter

Computer Vision to detect and count people by Iury Melo

Count the number of people using computer vision

This project uses YoloV3 to detect and count the number of people in a frame or image. It is made using Python 3.8, Flask, and OpenCV. Take in mind this project uses a free version of Heroku as host. It means sometimes the app may be slow to load, please be patient.

You have two options to use this project. The first is to use a live feed provided here. The feed shows an ice skating rink located in Moscow, Russia. Depending on the time of the day, it may be empty. The system will take a few frames, select one, and do its job.

The second option is to send a picture. If there is any person, it should detect and tell you the number of people detected in the image.
